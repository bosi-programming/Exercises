PIZZA PARTY

Write a program to divide pizzas. It should divide the pizzas evenly and, if
there are leftovers, it should display the number of leftover slices. Each
pizza has 8 slices.

Example:

How many people? 9
How many pizzas do you have? 2

9 people with 2 pizzas
Each person gets 1 piece of pizza.
There are 7 leftover pieces.
